# Icon Placeholder - icon16.png

Untuk sementara, Anda perlu membuat file icon16.png (16x16 pixels) untuk ekstensi ini.

## Quick Icon Creation

### Option 1: Simple Text Icon
1. Buka tool design seperti Figma, Canva, atau GIMP
2. Buat canvas 16x16 pixels
3. Background: Gradient biru (#4285f4 to #34a853) 
4. Tambahkan huruf "F" putih di tengah
5. Export sebagai PNG dengan background transparent

### Option 2: Form Icon
1. Buat canvas 16x16 pixels
2. Background: Biru (#4285f4)
3. Tambahkan icon formulir sederhana (garis-garis putih)
4. Export sebagai PNG

### Option 3: Download from Icon Libraries
- **Flaticon**: https://www.flaticon.com/search?word=form
- **Icons8**: https://icons8.com/icons/set/form
- **Heroicons**: https://heroicons.com/

### Option 4: Use Emoji as Temporary Icon
Untuk testing cepat, Anda bisa:
1. Screenshot emoji üìù atau üìã
2. Resize ke 16x16 pixels
3. Save sebagai icon16.png

## Required Files
- icon16.png (16x16) - Toolbar icon
- icon32.png (32x32) - Medium size  
- icon48.png (48x48) - Extension management
- icon128.png (128x128) - Chrome Web Store

Untuk development awal, Anda bisa copy icon16.png ke semua ukuran lain dengan rename.

## Installation Note
Extension tidak akan load tanpa minimal icon16.png. Pastikan file ini ada sebelum mencoba install extension.
